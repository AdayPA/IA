El codigo se encuentra en el archivo comprimido así como en el repo https://github.com/AdayPA/IA.
Para ejecutarlo:
  clonar el repo
  ir a la carpeta Practica_1_test
  compilar con make (hacer make clean si fuera necesario)
  ejecutar como ./main_maze
  insertar el archivo en el menu

Para utilizar el modo obstáculos aleatorios, previamente se tiene que insertan un archivo (dentro del menú).
